# LogicaMirror

LogicaMirror is a constrained reading and reasoning environment for predictive active recall.

The product is built around one loop:

```text
Read -> Predict -> Verify -> Review -> Reflect
```

The app must keep the learner's reasoning primary. AI is only a mirror for checking logic against gated source content.

# How to run
git clone https://github.com/9000wfxGit/LogicaMirror.git
cd LogicaMirror/LogicaMirror-main
node scripts/serve.mjs
